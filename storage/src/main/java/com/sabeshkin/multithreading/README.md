# Многопоточность

## RaceCondition

Проблема была в том, что join вызывался не в той же итерации, в которой запускаем тред.

## Deadlock

Проблему можно решить двумя способами: 
а) ошибка в том, что join вызывался уже после того как выполнили start второго потока.
б) либо поменять местами блокировки, тогда поток будет выбираться случайным образом.
