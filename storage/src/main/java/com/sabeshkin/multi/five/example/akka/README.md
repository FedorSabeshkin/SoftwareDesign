# AKKA

Использую фрэймворк akka для передачи двух типов сообщений и выборе нужного обработчика по типу сообщения.

```log
До свидания, Петрова
До свидания, Петров
До свидания, Козлов
До свидания, Чечевицин
Доброго утра Вам, гражданочка/гражданин, Петрова
Доброго утра тебе, Петров
Доброго утра Вам, гражданочка/гражданин, Козлов
Доброго утра тебе, Чечевицин
```
